# Sales Data Analysis Report

## Project Overview
This project analyzes a sales dataset using Python and the pandas library. The objective is to extract meaningful insights such as total sales, average sales, and the best-selling product.

## Dataset Exploration
The dataset was loaded from a CSV file and explored to understand its structure, including the number of rows, columns, and data types.

## Data Cleaning
- Missing values were handled by replacing them with zero.
- Duplicate records were removed to ensure accurate analysis.

## Metrics Calculated
The following metrics were calculated for all numeric columns:
- Total (Sum)
- Average (Mean)
- Maximum value
- Minimum value

Additionally, the best-selling product was identified based on total sales.

## Insights
The analysis provides a clear overview of sales performance and highlights the product contributing the most to revenue. These insights can help support better business decisions.

## Conclusion
This project demonstrates basic data analysis skills such as loading data, cleaning data, calculating statistics, and presenting results in a structured report.
